## 使用帮助

为了能在您的脚本中从'yoni-mcscripts-lib'中导入内容，你需要将编译后的文件手动复制到你的脚本目录中。

To use 'yoni-mcscripts-lib' as a runtime, you must copy the compiled files into 'scripts/' directory manually.
